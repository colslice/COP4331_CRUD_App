# Requirements

## App Functionality
- Upon initial access of the website, users can either login or sign up 
- Once logged in with a valid account, users can perform standard CRUD operations
  - Create
  - Read
  - Update
  - Delete
- Users can also search for contacts

- Minimum contact information for each contact record:
  - Name (prefer first name/last name)
  - Email
  - Phone
  - Date record created

## Technical
- Use a LAMP stack
- Use API for frontend to backend connection
- Create a Use Case diagram and a Gantt chart
- Use SwaggerHub for API documentation
- Create a Entity Relationship Diagram
